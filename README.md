# FLOAT

**Your money. Any chain. Just send.**

A chain-abstracted social money layer — one product, four modes (Send · Split · Leash · Pledge), zero chain awareness for the user. Built for the UXmaxx hackathon on Particle Network Universal Accounts (EIP-7702), Magic Labs embedded wallets, and Arbitrum One.

## Stack

Next.js 16 (App Router, Turbopack) · React 19 · TypeScript · Tailwind v4

## Getting started

```bash
npm install
cp .env.example .env.local   # UI runs without keys filled in
npm run dev                  # http://localhost:3000
```
